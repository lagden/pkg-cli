# @tadashi/pkg-cli

Update your `package.json` with mastery.

View on JSR: https://jsr.io/@tadashi/pkg-cli

## Install

```
npm i -g @tadashi/pkg-cli
```

## Usage

Go to your project folder:

```
npx @tadashi/pkg-cli
```

or

```
pkg-check
```

> [!NOTE]\
> To run `pkg-check` , you need to install it first.
